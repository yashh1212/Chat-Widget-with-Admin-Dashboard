Chat Widget Integration Example

How to Embed the Chat Widget

Add the following script tag to your website's HTML before the closing body tag:

<script src="http://localhost:5000/widget/chat-widget.js"></script>

Example Website

This is an example page showing how the chat widget will appear on your website. The chat bubble should appear in the bottom right corner of the page.

Click on the chat bubble to open the widget and start a conversation.

Features

  • Real-time messaging with Socket.IO
  • AI-powered responses using Google's Gemini API
  • Chat history persistence across page refreshes
  • Responsive design for all device sizes
  • Error handling for connection issues
